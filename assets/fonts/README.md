## 폰트 설치 안내 (Rama Gothic M)

이 프로젝트는 `Rama Gothic M` 타이틀 폰트를 웹폰트로 로드합니다.

현재 프로젝트에는 아래 OTF 파일이 있는 것으로 확인됩니다.

- `fonnts.com-Rama_Gothic_M_Bold.otf`

브라우저에서 OTF도 웹폰트로 로드할 수 있어, 위 파일만으로도 `Rama Gothic M`이 적용됩니다.

추가로 더 가볍고 빠른 로딩을 원하면 아래 파일도 함께 넣어주세요(선택).

- `RamaGothicM.woff2` (권장)
- `RamaGothicM.woff` (fallback)

WOFF2/WOFF가 존재하면 CSS가 우선적으로 사용합니다.
